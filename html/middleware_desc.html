미들웨어 셋팅

app.get( "/", (req, res, next) => { // res.send("익스프레스 서버 첫 메시지"); res.sendFile(path.join(__dirname, "index.html")); next(); }, (req, res) => { throw new Error("에러는 에러 처리 미들웨어로 갑니다."); } );

next() 호출하면 다음 미들웨어로 간다.

미들웨어 + URL

app.use(미들웨어) :: 모든 요청에서 미들웨어 실행

app.use('/abc', 미들웨어) :: abc로 시작하는 요청에서 미들웨어 실행

app.post('/abc', 미들웨어) :: abc로 시작하는 POST 요청에서 미들웨어 실행

app.use('/abc' 미들웨어)에서 app.post('/abc',미들웨어)로 이동

app.use("/abc", (req, res, next) => { console.log('next()함수로 호출해줘야 다음 미들웨어로 이동'); next(); });

에러 처리 미들웨어 :: 매개변수가 err, req, res, next로 네 개

실무에서는 직접 에러 처리 미들웨어를 연결

에러 처리 미들웨어는 특별한 경우가 아니면 가장 아래에 위치
